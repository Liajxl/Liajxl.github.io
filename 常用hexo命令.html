使用hexo编写个人博客

写入新博客

1、创建新的博客

hexo new HI创建标题为HI的博客，默认布局为post，默认类型可修改
hexo new "hi li"标题含有空格时，使用引号包围
hexo new page Hi创建标题为Hi的博客，布局设置为page

2、清理缓存

hexo clean清除缓存文件（db.json）和已生成的静态文件（public）

3、生成博客

hexo generate或 hexo g生成静态文件

4、部署网站

hexo deploy或 hexo d将静态文件部署到网站上
